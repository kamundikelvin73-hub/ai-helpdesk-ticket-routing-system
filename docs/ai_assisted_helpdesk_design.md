# AI-Assisted Helpdesk Design

## Goal

Use AI or rule-based classification to route support tickets faster and improve response consistency.

## Workflow

1. User submits ticket.
2. System extracts subject and description.
3. Ticket is classified by category.
4. Priority is assigned.
5. Ticket is routed to the correct team.
6. Reviewer checks uncertain cases.

## Human-in-the-Loop Review

AI should not make final decisions for high-risk security incidents without human review.

## Benefits

- Faster routing
- Reduced manual triage
- Better reporting
- More consistent support operations
